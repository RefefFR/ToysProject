Toy API Documentation

Documentation for the Toy API endpoints.

Base URL

        https://shook.cyclic.cloud

Endpoints

GET Requests

Get All Toys

        GET /toys

Returns a list of all toys.

Example Request:

GET http://localhost:3001/toys

Example Response:

            {
                "_id": {
                  "$oid": "65088de4787d464475e7ef70"
                },
                "name": "Fisher-Price Laugh & Learn Smart Stages Chair",
                "info": "An interactive chair with songs, phrases, and activities for toddlers.",
                "category": "Learning & Educational Toys",
                "img_url": "https://example.com/fisher_price_chair.jpg",
                "price": 29.99,
                "user_id": "65088bc2fe2865d1b6d70a73",
                "createdAt": {
                  "$date": "2023-09-18T17:50:28.352Z"
                },
                "updatedAt": {
                  "$date": "2023-09-18T17:50:28.352Z"
                },
                "__v": 0
              },

Search by Name or Info

        GET /toys/specify/?s={name or info}&page={pageNumber}

Search for toys by name or info.

Example Request:

GET http://localhost:3001/toys/specify/?name=Toy1&info=Info1

Example Response:

[
    {
        "_id": "123456",
        "name": "Toy1",
        "info": "Info1"
    }
]

Get Toys by Category

        GET /toys/category/?category={categoryName}&page={pageNumber}

Get toys by category.

Example Request:

GET http://localhost:3001/toys/category/?catname=Category1

Example Response:

[
    {
        "_id": "123456",
        "name": "Toy2",
        "category": "Category1"
    },
    {
        "_id": "789012",
        "name": "Toy3",
        "category": "Category1"
    }
]

Filter by Price Range

        GET /toys/prices/?min={minPrice}&max={maxPrice}

Filter toys by price range.

Example Request:

GET http://localhost:3001/toys/prices/?min=20&max=50

Example Response:

[
    {
        "_id": "123456",
        "name": "Toy4",
        "price": 30
    },
    {
        "_id": "789012",
        "name": "Toy5",
        "price": 40
    }
]

Get Toy by ID

        GET /toys/{id}

Get a toy by its ID.

Example Request:

GET http://localhost:3001/toys/123456

Example Response:

{
    "_id": "123456",
    "name": "Toy6",
    "info": "Info6"
}

POST Requests

Create a New Toy

        POST /toys

Create a new toy. Requires authentication.

Example Request:

POST http://localhost:3001/toys

Example Response:

{
    "_id": "123456",
    "name": "New Toy",
    "info": "New Toy Info",
    "img_url":"https://img_url"
}

img_url is not a must

PUT Requests

Update a Toy

        PUT /toys/{id}

Update a toy by ID. Requires authentication.

Example Request:

PUT http://localhost:3001/toys/123456

Body:

            {
                "_id": "123456",
                "name": "New Toy",
                "info": "New Toy Info",
                "img_url":"https://img_url"
            }

Example Response:

{
    "message": "Toy updated successfully"
}

DELETE Requests

Delete a Toy

        DELETE /toys/{id}

Delete a toy by ID. Requires authentication.

Example Request:

DELETE http://localhost:3001/toys/123456

Example Response:

{
    "message": "Toy deleted successfully"
}
